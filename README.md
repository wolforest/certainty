# 确定性 & [猜测](./guess.md)
AI时代，普通人并不适合乘风破浪。<br />
在不确定的时代，依赖确定性、通过学习构筑竞争优势，是普通人的生存之道。<br />
这一点对于软件工程师尤为重要，这个项目尝试收集AI时代的确定性及大概率的猜测。<br />
希望能同广大软件工程师一起成长、一起面对未来的挑战。

## AI & Software
* 软件还是软件，用于解决真实的问题。AI不会改变这一点。
* AI时代软件多了一种用户：AI agent。
* AI时代，更多复杂的算法得到广泛应用。
* 持续学习是AI时代的基本要求。

## 职业规划
* 开源、读书、写作是软件工程师的确定性。
* 参与顶级软件开发，能力与职业双受益。
* 读书、尤其是读经典著作是软件工程师的基本功。
* 体系化写作是软件工程师的核心竞争力，是AI不具备的能力。

## 软件行业
* 基于云的软件生态会继续发展。
  * 更丰富的PAAS生态
  * 更丰富的SAAS生态
* 软件质量的诉求与软件的生命周期有关。
